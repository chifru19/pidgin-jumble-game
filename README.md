# 🎮 Pidgin Jumble Game: Oga Edition

Welcome to the **Original Pidgin Jumble Game**! This is a fun, educational word scramble game built entirely in Python. Unlike standard word games, this one uses **100% Pidgin English** logic to test your "sabi" levels.

## 🚀 Features
- **Level System:** Start from "Easy" and work your way up to "Ogbonge" (Hard).
- **Dynamic Scoring:** Earn more points as the words get longer and the levels get tougher.
- **Pidgin Logic:** Designed for the culture, by the culture.

## 🛠️ How to Play
1. Make sure you have **Python** installed on your computer.
2. Download the `main.py` file from this repository.
3. Open your terminal or command prompt.
4. Run the game using:
   ```
   bash
   python main.py
   ```
