## ✨ Features
* **Authentic Vocabulary:** Test your "sabi" with words like *WAHALA*, *OYIBO*, *CHOP*, and *KPAKO*.
* **Lives System:** You start with 3 hearts—one wrong move and you lose a life!
* **Dynamic Hints:** Spend 5 points to get a "Help Me Small" hint.
* **Visual Feedback:** Arcade-style screen flashes and custom UI effects.
* **Local Leaderboard:** Persistent high-score tracking saved to `leaderboard.txt`.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI Library:** Tkinter
* **Audio Engine:** Pygame (for immersive sound effects)
* **Assets:** Custom graphics processed via Pillow
