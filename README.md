## 💻 Local Setup
1. **Clone:** `git clone https://github.com/chifru19/pidgin-jumble-game.git`
2. **Install:** `pip install pygame pillow`
3. **Run:** `python gui_game.py`

## ✨ Features
* **Authentic Vocabulary:** Words like *WAHALA*, *OYIBO*, *CHOP*, and *KPAKO*.
* **Lives System:** 3 hearts—don't lose them!
* **Dynamic Hints:** Spend 5 points for a "Help Me Small" hint.
